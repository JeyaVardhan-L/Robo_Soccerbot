# Competition Rules Summary

This robot was built under the following constraints:

## Mechanical Constraints
- Max footprint: 40cm x 40cm
- Max height: 40cm
- Max weight: 4kg (target under 3kg)

## Motor Rules
- Only DC motors allowed
- Max motor speed: 300 RPM (no load)

## Ball Interaction
- No grabbing or trapping
- Only open pushing allowed
- No suction, no locking ball

## Wireless
- Must be locally controlled
- No cloud processing
- FlySky transmitter used

## Power
- Onboard battery only
- Max open circuit voltage: 12.8V
- 3S LiPo used (11.1V nominal)

## Safety
- Must have ON/OFF switch
- Must include failsafe
- No sharp edges
- No damaging mechanisms

## Arena
- 6m x 3m field
- Synthetic turf
- 20cm walls
- 60cm goal width
- 5 min match (2.5 min halftime)

This document guided all hardware and firmware decisions.
