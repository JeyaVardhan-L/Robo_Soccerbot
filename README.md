# Robo Soccer Bot

This repository documents the complete engineering journey of building a Robo Soccer bot using:

- Arduino UNO
- BTS7960 motor drivers
- FlySky transmitter
- 4 DC motors (300 RPM)
- 3S LiPo battery

This repo includes:
- Hardware architecture
- Wiring explanations
- Debugging process
- Firmware evolution
- Competition-ready final code
